# Quickstart

## 1) Launch
- **Deployed app**: open *WAHi* from Applications / Start menu.
- **MATLAB**: run:
  ```matlab
  wahi_gui
  
2) Create a geometry

Click 1 Random Config ♯ (or load your own CSV).

3) Define the Field of Accuracy (FoA)

In Array & Field Parameters:
	•	Set FoA outer radius (R) and inner radius (Rin)
	•	Set grid spacing
	•	Keep a sensible Min source–mic distance to avoid near-field degeneracies

Click Apply Changes.

4) Evaluate baseline

Click 2 Run WAH ▶︎

You should now see a 3D colour field + summary metrics (Pass, Mean, P95, Max).

5) Optimise

Click 3 Optimise ᯓ★

Start with coarse settings first (see Optimisation tips).
